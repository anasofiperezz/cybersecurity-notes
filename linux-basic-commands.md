# Linux Basic Commands

## What is Linux?

Linux is an operating system commonly used in cybersecurity, servers, and networking.

Many cybersecurity tools work on Linux systems.

---

# Basic Linux Commands

## pwd
Shows the current directory.

```bash
pwd
```

---

## ls
Lists files and folders.

```bash
ls
```

---

## cd
Changes directory.

```bash
cd Documents
```

---

## mkdir
Creates a new folder.

```bash
mkdir cybersecurity
```

---

## touch
Creates a new file.

```bash
touch notes.txt
```

---

## rm
Deletes files.

```bash
rm notes.txt
```

---

## clear
Clears terminal screen.

```bash
clear
```
