# Linux Practice Lab 01

## Objective

Practice basic Linux commands for navigation and file creation.

---

## Commands practiced

```bash
pwd
ls
mkdir cyber-lab
cd cyber-lab
touch notes.txt
ls
```

---

## What each command does

- `pwd`: shows the current directory.
- `ls`: lists files and folders.
- `mkdir cyber-lab`: creates a new folder.
- `cd cyber-lab`: enters the folder.
- `touch notes.txt`: creates a new file.
- `ls`: confirms the file was created.

---

## Reflection

In this lab, I practiced basic Linux commands used to navigate directories and create files. These commands are important for cybersecurity because many tools and servers use Linux environments.
